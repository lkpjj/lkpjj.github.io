python简易闹铃程序

直接看下面的程序实现：

在运行这个程序的过程中由于之前一点经验都没有还是碰到许许多多的问题，所以这里记载一下:
首先原文的作者导入的winsound，搞了半天才知道原来那个windows下面使用的，自己使用的linux，所以查了半天发现应该导入ossaudiodiv，然后就是要用到pyglet，由于我装了python3.3 和 python2.7，所以当通过**pip install pyglet**安装之后在python2的site-package里面并没有pyglet，因为它会装到你默认的python的lib，我就直接从python3拷贝了一份到python2下面，事实证明是完全可以用的；到了这里不要以为就完全可以运行了，运行的时候又出现这样的错误：** WAVEFormatException: AVbin is required to decode compressed media ** 错误，查询了半天找到了一个解决办法，可能由于是因为我使用的是64位的系统，所以去下载了**avbin-linux-x86-64-v8**,解压之后， sudo ./install.sh,然后才能正常运行，当闹铃响的时候你就会听到你在程序里设置的音频文件，Happying…….
