name的作用

之前看到一些python的程序中总会有这么一句：if name == “main:”
一直比较奇怪为什么加这么一句，查了一下才知道，原来这个东西是为了区别你运行的这个程序是直接运行的程序本身还是通过import 来导入之后运行的，可能这么说比较不清楚，还是直接看个程序比较明了：

命令:python Test.py
输出为：Hello World

然后我们使用import导入之后看下：

可以看出这个时候的name为你导入的程序的名称;然后我们直接输入看下输出

这个才是你正在运行的那个程序的name的值,大致的意思就是这个，可能表达起来有些不清楚了。。。。。。
