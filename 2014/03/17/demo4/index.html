####const

  1. const的用途

    • 可以定义const常量
    • const可以修饰函数的参数和返回值，甚至函数体
    • 被const修饰的东西受到强制保护，防止意外变动

  2. const与#define的区别

    • const常量有数据类型，而宏常量没有数据类型，前者编译器会进行类型检查，而后者只是进行字符替换
    • 有的工具可以调试const常量，但不能调试宏常量

####sizeof
不同数据类型所占字节(32位编译器)：

  • 指向任何数据类型的指针所占字节数都是4
  • char ：1个字节
  • char*（即指针变量）: 4个字节
  • short int : 2个字节
  • int： 4个字节
  • unsigned int : 4个字节
  • float: 4个字节
  • double: 8个字节
  • long: 4个字节
  • long long: 8个字节
  • unsigned long: 4个字节

  • Code

  • Result(arch 64bit)

  • Tips

float f占4个字节，char p占一个字节，int temp[3]占12个字节，总共是17个字节，依据*内存对齐的原则*，要选择4的倍数，因此结果是20个字节。

  • 那么一个空类和一个多重继承的空类分别占多少空间呢？
  1. 空类所占空间为 1
  2. 单一继承的空类所占空间为 1
  3. 多重继承的空类空间还是 1
  4. 虚继承涉及到虚指针，所以空间应该为 4
