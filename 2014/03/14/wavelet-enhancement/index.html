matlab中小波分解介绍

  • wavelet
matlab

前几天看了一篇做关于小波增强图像的文章，作者用到了小波分解，对于图像而言，很显然要用到二维的小波分解，因此这里主要介绍二维的小波分解，至于一维的分解过程读者可以参看其他文章;
爱迪生发生地方噶地方

####二维小波分解

######函数名 函数功能

  • dwt2 二维离散小波变换，实现单层的小波分解
  • idwt2 二维离散小波分解反变换
  • wavedec2 二维小波信号的多层分解
  • waverec2 二维信号的多层小波重构
  • wrcoef2 由多层小波分解重构某一层的分解信号
  • upcoef2 由多层小波分解重构近似分量或细节分量
  • detcoef2 提取二维信号小波分解的细节分量
  • appcoef2 提取二维信号小波分解的近似分量
  • upwlev2 二维小波分解的单层重构
  • dwtpet2 二维周期小波变换
  • idwtper2 二维周期小波反变换

######wavedec2

  1. 功能：该函数是二维的小波分解函数,即多尺度分解
  2. 用法：[C,S] = wavedec2(X,N,’wname’) | [C,S] = wavedec2(X,N,’wname’)
  3. 说明：对图像X用wname小波基实现N层分解

输出向量C的实际存储方式为：

  • A = approximation coefficients
  • H = horizontal detail coefficients
  • V = vertical detail coefficients
  • D = diagonal detail coefficients

他们每一个都是一个行向量，分别存储的是各级小波的系数

输出向量S的存储方式为：
S(1,:) = size of approximation coefficients(N).
S(i,:) = size of detail coefficients(N-i+2) for i = 2, …N+1 and S(N+2,:) = size(X).

######waverec2

  1. 功能：该函数是二维的小波分解重构函数
  2. 用法：
    • X = waverec2(C,S,’wname’)
    • X = waverec2(C,S,Lo_R,Hi_R)
    • X = waverec2(wavedec2(X,N,’wname’),’wname’)
    • X = waverec2(C,S,’wname’)
    • X = appcoef2(C,S,’wname’,0)
      C，S的意义同wavedec2
  3. Example

官方文档

######appcoef2
二维离散小波变换低频系数提取

  • A = appcoef2(C,S,’wname’,N)
  • A = appcoef2(C,S,’wname’)
  • A = appcoef2(C,S,Lo_R,Hi_R)
  • A = appcoef2(C,S,Lo_R,Hi_R,N)
    假如我们要抽取一级小波的低频分量，抽取结果为矩阵，尺度见尺度矩阵S
    LL1 = appcoef2(c,s,'wname',1)

######detcoef2
二维小波分解变换高频系数提取
D = detcoef2(O,C,S,N)

####小波系数的物理意义
一直不太理解小波系数的意义，下面这段话个人觉得描述的比较通俗易懂一些，所以拿来跟大家分享一下:
“小波变换和傅立叶变换一样，都是将信号在一基中展开，从而体现信号的特征，在不同基中展开，我们将从不同的侧面得到信号的相应特征，或者说，基的选取与我们所关心的信号特征有关，好的基意味着我们可以用较少的系数（或基坐标）来表示信号。比如，用傅立叶基我们关心的是信号的频率成分，而用小波基，我们更关心的是信号的局部特征（这也是为什么小波基具有紧支撑性或叫做“小波”的原因）。信号在基中展开，其系数就是信号与不同基函数的内积，通俗的说就是信号在基坐标上所对应的值（或者坐标），这与二维、或三维坐标的含义一样。”原文
